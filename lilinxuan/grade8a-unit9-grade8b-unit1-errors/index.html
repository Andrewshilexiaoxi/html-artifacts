一、单项选择 · 第3题

( )3. —I don't know how to use the APP Fun Dubbing.
—Ask Jimmy for help. He has a lot of in doing it.

A. experience
B. trouble
C. courage
D. agent
✅ 正确答案：A

📝 考点分析：名词词义辨析

1. 题目句意："——我不知道怎么使用Fun Dubbing这个应用程序。——找Jimmy帮忙，他在这方面很有经验。"

2. 选项分析：

- A. experience 经验，不可数名词，have a lot of experience in doing sth 表示"在做某事方面有很多经验"，语法正确，语义完全符合语境，是正确答案

- B. trouble 麻烦，have trouble in doing sth 表示"做某事有困难"，既然让对方去找Jimmy帮忙，就不可能说他做这件事有很多麻烦，不符合语境

- C. courage 勇气，虽然语法正确，但语义不通。这句话表达的是"因为Jimmy有经验所以找他帮忙"，而不是"因为他有勇气所以找他帮忙"

- D. agent 代理人，词义完全不相关

一、单项选择 · 第6题

( )6. —Hello, Sunshine Restaurant! What can I do for you?
—Do you have a table for this Sunday?

A. available
B. useful
C. empty
D. possible
✅ 正确答案：A

📝 考点分析：形容词词义辨析（交际用语 - 餐厅预订）

1. 题目句意："——你好，阳光餐厅！请问需要什么帮助？——你们周日有可预订的空位吗？"

2. 选项分析：

- A. available 可获得的、可预订的，a table available 表示"有空位可供预订"，这是餐厅预订场景中的标准常用表达，正确答案

- B. useful 有用的，不用于描述餐厅是否有空桌

- C. empty 空的，empty table 指桌子现在没有坐人，但在预订语境中，客人问的是能否预订，不是问桌子现在是不是空着，所以available更准确

- D. possible 可能的，语义不通

一、单项选择 · 第8题

( )8. —May I use your computer?
—Sorry, I'm halfway to my online homework.

A. do
B. doing
C. did
D. have done
✅ 正确答案：B

📝 考点分析：非谓语动词 - 固定搭配

1. 题目句意："——我可以用你的电脑吗？——对不起，我的在线作业才做到一半。"

2. 关键语法点：be halfway to ... 是固定搭配，这里的 to 是介词，不是不定式符号，因此介词后必须接名词或动名词（doing）形式

3. 选项分析：

- A. do 动词原形，语法错误，to作为介词后不能接动词原形

- B. doing 动名词，be halfway to doing sth 是英语中固定正确用法，表示"某事做了一半"，正确答案

- C. did 过去式，语法错误

- D. have done 现在完成时，语法错误，这里不需要谓语动词，to是介词不是不定式符号

二、完形填空 · 第2题

📖 原文语境

"Well," said Eliza, "I'd rather go with you and Mom."

"We hope you can come with us, too. ___2.___ we're going to be so busy that you won't have any fun. Last week, you mentioned a football camp and Cindy would attend it. Will you ___1.___?" Dad asked.

( )2. 选出正确选项：

A. Or
B. So
C. But
D. And
✅ 正确答案：C

📝 考点分析：连词词义辨析（逻辑关系）

1. 上下文逻辑：前文说"We hope you can come with us, too."（我们也希望你和我们一起去），后文说"we're going to be so busy that you won't have any fun."（我们会很忙，你不会玩得开心）。

2. 前后文是转折关系：虽然希望你去，但是因为太忙，你可能玩得不开心，所以用but表示转折。

3. 选项分析：

- A. Or 或者，表示选择

- B. So 因此，表示因果

- C. But 但是，表示转折，符合逻辑

- D. And 和，表示并列
