Andrew's English Teaching Pages

英语教学 HTML 资料库

这里用于发布给学生查看的 HTML PPT、语法讲解、错题复盘、阅读精讲和互动练习。每份资料都可以放在独立文件夹里，生成一个可以直接发给学生的网页链接。

https://andrewshilexiaoxi.github.io/html-artifacts/
适合放什么
  • 给单个学生的 HTML PPT
  • 一节课的语法复习页面
  • 阅读精讲和词汇复盘
  • 不含隐私的错题讲解
每个学生一个链接

把一份 HTML PPT 放进一个单独文件夹，并命名为 index.html。学生收到文件夹对应的网址后，打开就是自己的学习页面。

隐私边界

GitHub Pages 是公开网页。建议使用学生代号、日期、课程主题命名，不要放真实姓名、成绩、联系方式或敏感评价。

1. 做好 HTML 把课件导出或保存为一个完整的 HTML 页面。
2. 建文件夹 用学生代号、日期和主题命名，避免暴露隐私。
3. 上传文件 文件夹内主页面统一叫 index.html。
4. 发送链接 把对应文件夹的网址发给学生本人即可。 
html-artifacts/
  index.html
  s20260511-grammar-review/
    index.html
  s20260511-reading-notes/
    index.html
  class-vocab-lesson-01/
    index.html
推荐链接格式

/s20260511-grammar-review/

例如：
https://andrewshilexiaoxi.github.io/html-artifacts/s20260511-grammar-review/
